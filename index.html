Projects

In developing a Python-based Tic Tac Toe game, I designed an engaging yet straightforward console application that allows two players to compete in a timeless grid-based strategy game. Utilizing Python's intuitive syntax, the game's core was structured around a series of functions managing game logic, player input, and display updates. Players alternate turns, inputting their moves via simple numeric keypad positions, which correspond to specific cells in a 3x3 grid. The program continuously checks for winning conditions—three consecutive identical symbols in a row, column, or diagonal—or the scenario where all cells are filled, resulting in a draw. Enhancements such as an AI opponent or graphical interface could be considered for future versions, but the current iteration focuses on solidifying the fundamental gameplay mechanics and ensuring a seamless user experience. The user must use these controls. Left Mouse Click: Make a move. 'G': Toggle game mode between AI and PvP.'R': Restart the game.'0': Set AI to random mode. '1': Set AI to use the minimax algorithm
